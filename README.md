# 声が届く入試設計 (Voice Admission System)

未踏ジュニア2025応募用のプロトタイプコードです。
入試制度のドキュメントにAIチャットボットを組み合わせ、ユーザーの疑問を解決（質問モード）しつつ、制度に対する建設的な改善案を収集（提案モード）するWebアプリケーションです。

## 主な機能
* **質問モード (RAG):** ドキュメントの内容に基づき、入試制度に関するユーザーの質問にAIが回答します。
* **提案モード:** ユーザーの改善提案から「対象」「要約」「感情」を自動抽出し、データベース（想定）へ構造化して送信します。

## 使用技術 (Tech Stack)
* Python
* Streamlit (フロントエンドUI)
* LangChain / LLM API (自然言語処理)

※セキュリティのため、`.env`ファイル（APIキー等）およびローカルのデータベース接続設定は本リポジトリから除外しています。
